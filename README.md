# AICTE

Go to 'gh-pages' branch to see just the static pages.
and 'master' branch for development.

## Installation

```
npm run setup 
 ```

### Run on Development environment
 ```
npm run start
 ```
 
### Run on production environment
 ```
npm run start:dist
 ```

### build dist folder & push into gh-pages branch
 ```
npm run bd
 ```
